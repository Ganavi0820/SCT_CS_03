# Password Strength Checker

This is a Python program that checks the **strength of a password** based on several criteria such as length, presence of uppercase and lowercase letters, numbers, and special characters. It provides feedback to help users create stronger, more secure passwords.

---

## Features

- Checks if the password has **at least 8 characters**.
- Verifies the presence of **lowercase letters**.
- Verifies the presence of **uppercase letters**.
- Verifies the presence of **numbers**.
- Verifies the presence of **special characters** (`!@#$%^&*(),.?":{}|<>`).
